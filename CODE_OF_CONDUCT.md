# Code of Conduct

## Our Standard

This repository is maintained as a public automation template and portfolio project. Discussions should be respectful, practical, and focused on improving the system.

Examples of good behavior:

- clear issue reports;
- constructive pull request feedback;
- respectful technical disagreement;
- safe examples without real lead data;
- useful privacy, security, and automation improvements.

## Unacceptable Behavior

Unacceptable behavior includes:

- harassment or personal attacks;
- publishing tokens, database passwords, webhook URLs, real leads, customer contacts, or private business data;
- spam;
- abusive language;
- irrelevant promotion;
- requests to misuse customer data or bypass consent requirements.

## Enforcement

Maintainers may edit, close, delete, or lock issues and pull requests that violate this code of conduct.

## Reporting

Report problems through GitHub issues with a safe, factual description. Do not include private lead data, credentials, webhook URLs, or personal information.
