# Maintenance Notes

This file keeps lightweight maintenance rules for Telegram Lead Capture System.

## Regular Checks

- Keep website widget examples generic.
- Keep Telegram bot examples free from real chats.
- Review n8n webhook assumptions after workflow changes.
- Keep PostgreSQL schema notes current.
- Update privacy notes when lead-processing behavior changes.

## Safe Data Rules

Use placeholders for Telegram tokens, chat IDs, webhook URLs, database passwords, customer names, phone numbers, email addresses, and real lead content.

## Release Hygiene

Before publishing a new update, check README accuracy, setup steps, architecture notes, security policy, support guide, and demo placeholders.
